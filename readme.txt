Classic Widgets for WP will will restore classic mode of widgets under Appearance -> Widgets and under Customizer -> Widgets.

== Description ==

WordPress 5.8 introduced new widgets but many users are not familiar with new widgets or may want to revert to older classic mode of widgets then thhis plugin will help.

On activation, plugin will enable old classic widgets under Appearance -> Widgets as well under Customizer -> Widgets. There is no extra settings.
